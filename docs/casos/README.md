# Documentación de casos — Migraciones DGSISAN

Esta carpeta está pensada para alojar documentación de las migraciones realizadas en el ámbito de salud digital (SIGEHOS / DGSISAN).

## Cómo agregar tus documentos locales

Copia desde tu máquina los archivos de:

```
F:\DOCUMENTOS\dgsisan\laboratorio\migraciones
```

hacia:

```
docs/casos/laboratorio/
```

Estructura sugerida:

```
docs/casos/
├── laboratorio/          # Planes, arquitectura, diagramas
├── gestor-encuentro/     # Documentación GDE
└── bus-interoperabilidad/ # Documentación del bus
```

> **Nota:** No subas datos sensibles, credenciales ni información clínica identificable. Para el portafolio público bastan resúmenes técnicos y diagramas anonimizados.
